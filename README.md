# silver-disco

## Стоит (Worth It)

A simple utility to check if something is worth its cost.

### Usage

Run the program:

```bash
python3 stoit.py
```

The program will prompt you to enter a value and a cost, then tell you if it's worth it.

### Features

- Simple value vs. cost comparison
- Bilingual output (Russian/English)
- Interactive command-line interface
## Проверка связи / Connection Check

Простая веб-страница для проверки интернет-соединения.

A simple web page to check internet connectivity.

### Особенности / Features

- ✓ Автоматическая проверка при загрузке страницы / Automatic check on page load

- ✓ Отслеживание изменений статуса подключения / Real-time connection status monitoring
- ✓ Двуязычный интерфейс (RU/EN) / Bilingual interface (RU/EN)
