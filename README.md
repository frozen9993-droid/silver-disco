# silver-disco

## Стоит (Worth It)

A simple utility to check if something is worth its cost.

### Usage

Run the program:

```bash
python3 stoit.py
```

The program will prompt you to enter a value and a cost, then tell you if it's worth it.

### Features

- Simple value vs. cost comparison
- Bilingual output (Russian/English)
- Interactive command-line interface
